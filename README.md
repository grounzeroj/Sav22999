> koa-cli-man,一款基于nodejs的新的web框架koa示例代码生成脚手架.
 
`koa-cli-man`,采用的package.json配置`"type":"module",`使用esm规范.

也可默认为`"type": "commonjs",`采用的commonjs模块规范.

## Prerequisites

- node >13.0.0

## Install

```sh
yarn
yarn install
或者
npm i
npm install
```

## npm或yarn全局安装与使用

`全局npm安装或yarn安装`

```sh
npm i -g koa-cli-man
或者
npm install -g koa-cli-man
或者 TODO yarn
yarn add global koa-cli-man
```

`全局命令使用` `使用下面命令`

```sh
koa-cli-man
或者
kcman
或者
koa-cli
```

## Usage

```sh
yarn app
或者
npm run app
```
